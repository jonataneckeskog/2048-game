# 2048 Game

A desktop version of the classic **2048 puzzle game**, built with Java and Swing.

## Gameplay
- Slide the tiles using arrow keys.
- Combine tiles of the same value to create higher-value tiles.

---

## Requirements

- Java 21
- Maven 3.x

---
